PLUGIN OVERVIEW
================================================================================

Friends of Naples Gestionale is a comprehensive WordPress plugin for managing
non-profit organizations, associations, and clubs. It provides complete member
management, payment tracking, fundraising campaigns, and administrative tools.

================================================================================
CUSTOM POST TYPES
================================================================================

1. fg_socio (Members/Soci)
   - Title: Member name
   - Meta Fields: fiscal code, email, phone, address, subscription date,
                  expiry date, annual fee, status, notes
   - Taxonomy: Member categories
   - Supports: title, editor, thumbnail
   - Menu Icon: dashicons-groups

2. fg_pagamento (Payments/Pagamenti)
   - Title: Payment reference
   - Meta Fields: member ID, amount, payment date, payment method,
                  payment type, notes
   - Taxonomy: Payment status
   - Supports: title
   - Menu Icon: dashicons-money-alt

3. fg_raccolta (Fundraising/Raccolte Fondi)
   - Title: Campaign name
   - Meta Fields: goal, collected amount, start date, end date, status
   - Supports: title, editor, thumbnail
   - Menu Icon: dashicons-heart
   - Public: Yes (visible on frontend)

================================================================================
KEY FEATURES
================================================================================

MEMBER MANAGEMENT
- Complete profile with personal and contact information
- Document upload and management
- Automatic expiry date calculation
- Multiple status types (Active, Suspended, Expired, Inactive)
- Category classification

PAYMENT TRACKING
- Link payments to specific members
- Multiple payment methods (Cash, Bank Transfer, Card, PayPal)
- Payment types (Membership, Donation, Event)
- Payment history and reporting

FUNDRAISING CAMPAIGNS
- Goal setting and progress tracking
- Visual progress bars with percentage
- Campaign status management
- Public display with shortcodes
- Featured images support

ADMINISTRATIVE DASHBOARD
- Real-time statistics (total members, active, expired, revenue)
- Interactive charts (Chart.js)
  * Payment trends (12 months line chart)
  * Member distribution (pie chart)
- Recent payments display
- Active campaigns monitoring
- Quick action buttons

REMINDER SYSTEM
- Daily automated checks
- Configurable reminder period (default: 30 days)
- Automatic status updates for expired memberships
- Admin dashboard notifications
- Email notifications

EMAIL NOTIFICATIONS
- Welcome email for new members
- Expiration reminder emails
- Payment confirmation emails
- Customizable templates
- Dynamic placeholders ({nome}, {data_scadenza}, {quota})

DATA EXPORT
- CSV export for all data types
- UTF-8 encoding with BOM (Excel compatible)
- Filters for members (by status)
- Filters for payments (by date range)
- Filters for campaigns (by status)

SHORTCODES
1. [fg_elenco_soci] - Display members list
   Parameters: categoria, stato, limite, ordina, ordine

2. [fg_dettaglio_socio] - Display single member
   Parameters: id

3. [fg_elenco_raccolte] - Display fundraising campaigns
   Parameters: stato, limite

4. [fg_dettaglio_raccolta] - Display single campaign
   Parameters: id

5. [fg_progress_bar] - Display progress bar widget
   Parameters: raccolta_id

6. [fg_dashboard] - Display statistics dashboard
   Parameters: none (admin only)

7. [fg_filtro_soci] - Display member filter form
   Parameters: none

================================================================================
SETTINGS & CONFIGURATION
================================================================================

Admin → Friends Gestionale → Impostazioni:

1. Quota Annuale Predefinita (Default Annual Fee)
   - Type: Number (decimal)
   - Default: €50
   - Used for auto-populating member fees

2. Giorni Anticipo Reminder (Reminder Days)
   - Type: Number
   - Default: 30
   - Days before expiry to send reminder

3. Notifiche Email (Email Notifications)
   - Type: Checkbox
   - Default: Enabled
   - Enable/disable automatic emails

4. Oggetto Email Reminder (Reminder Subject)
   - Type: Text
   - Customizable email subject

5. Messaggio Email Reminder (Reminder Message)
   - Type: Textarea
   - Customizable email body
   - Supports placeholders

================================================================================
TROUBLESHOOTING GUIDE
================================================================================

ISSUE: Emails not sending
SOLUTION: Install WP Mail SMTP plugin, configure SMTP settings

ISSUE: Dashboard empty
SOLUTION: Check user role (must be Administrator), clear cache

ISSUE: Shortcodes not working
SOLUTION: Verify plugin is active, check shortcode syntax

ISSUE: Export fails
SOLUTION: Check server memory limit, try smaller datasets

ISSUE: Images not loading
SOLUTION: Check file permissions, verify upload directory

ISSUE: Cron not running
SOLUTION: Check WordPress cron is enabled, use cron plugin
